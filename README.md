# The Extend Theme Customizer

## Installation

```
git clone https://github.com/devyg/extend-theme-customizer.git extend-theme-customizer
```

## Usage

JSON file : See json/theme-customizer-settings-default.json

## Option Type

* text
* textarea
* text-editor
* color
* image
* layout-picker
* date-picker
* category-dropdown
* google-font
* menu-dropdown
* post-dropdown
* post-type-dropdown
* taxonomy-dropdown
* user-dropdown
